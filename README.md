# 🖼️ Fast Image Rotator

A high-performance, browser-native image rotation and transformation tool. Designed for users who need a fast, professional, and **100% private** way to edit images without uploading them to any server.

[![GitHub license](https://img.shields.io/github/license/rifatcholakov/image-rotate?style=flat-square)](https://github.com/rifatcholakov/image-rotate/blob/main/LICENSE)
[![PRs Welcome](https://img.shields.io/badge/PRs-welcome-brightgreen.svg?style=flat-square)](http://makeapullrequest.com)

## 🌟 Why Image Rotator Pro?

Most online image editors require you to upload your files to their servers, which can be a privacy risk. **Image Rotator Pro** uses HTML5 Canvas technology to process everything locally in your browser. Your images never leave your machine.

---

## ✨ Features

- **🛡️ Privacy First**: Zero server-side processing. All transforms happen on your local hardware.
- **🔄 Precision Transforms**:
  - **Fine rotation slider**: Adjust the angle from 0° to 360° with 1-degree precision.
  - **Quick Rotate**: Instant -90° and +90° buttons for standard adjustments.
  - **Mirroring**: One-click Horizontal and Vertical flipping.
- **🌗 Custom Themes**: Seamless switching between a sleek midnight glassmorphic theme and a clean light mode.
- **📊 Live Metadata**: Instant readout of image dimensions and auto-calculated orientation (Portrait/Landscape).
- **🎉 Premium Interactions**:
  - **Shimmer Effect**: Visual feedback during the rendering process.
  - **Confetti Celebration**: Sparkles on every successful download!
- **⚡ Performance**: Zero-lag preview using CSS transforms, followed by high-quality Canvas baking on export.

## 🛠️ Tech Stack

- **React 19**: modern, efficient UI rendering.
- **Vite 8**: ultra-fast development and build environment.
- **Lucide React**: consistent, high-quality iconography.
- **Canvas-Confetti**: delightful user feedback.
- **Vanilla CSS**: a custom-built, glassmorphic design system (no Tailwind bloat).

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/rifatcholakov/image-rotate.git
   ```

2. **Navigate to the directory:**
   ```bash
   cd image-rotate
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Launch the app:**
   ```bash
   npm run dev
   ```

The app will now be running at `http://localhost:5173/`.
